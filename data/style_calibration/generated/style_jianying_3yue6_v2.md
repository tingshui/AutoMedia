# Style: 剪映导入-3月6日 v2

## Scope

Vertical talking-head parenting/education opinion videos where the speaker moves from a child conflict to a parent-facing reframe and practical takeaway.

Status: `needs_review`.

Confidence: medium for videos similar to the 3月6日 reference, low for general creator style.

## Overall Feel

This is a口播 plus light-variety style. The voice remains primary, but the edit should actively mark the argument's emotional beats. The viewer should feel: hook, confusion, wrong instinct, better reframe, example mode, hidden seriousness, hurt, practical method, useful idea, completion.

## Editing Rules

| Rule ID | Rule |
|---|---|
| R1 | Keep speech primary. Every effect/audio cue must be tied to a content function. |
| R2 | In the first 10% of the video, use both a framing/focus visual cue and a question/confusion cue if the hook contains conflict or uncertainty. |
| R3 | Mark wrong-parent-instinct moments with error/negative audio. |
| R4 | Mark better-reframe moments with correct/success audio. |
| R5 | In example/recalled-interaction zones, use repeated tape/retro or focus effects every 8-12 seconds while the example remains active. |
| R6 | When a hidden social/emotional problem becomes explicit, use a tension or shock cue and, if the point lasts longer than one sentence, a glitch/disruption effect. |
| R7 | When the child's hurt or被处理/被欺负 consequence is centered, use a sad/broken cue and reduce comedic cues nearby. |
| R8 | At the start of practical advice, restart energy with alert, spotlight, urgency, or tape-style emphasis. |
| R9 | During dense explanation, maintain one light emphasis cue every 8-12 seconds, alternating visual and audio families. |
| R10 | Near the final 15% of the video, use idea/success/completion cues in sequence: useful idea, action achieved, final completion. |
| R11 | Use overlays briefly on hooks, shock/realization moments, instructions, and final flourish. Overlay duration should usually be 1-4 seconds. |
| R12 | Keep subtitles continuous and sentence-level throughout the speaking section. |

## Effect Vocabulary

| Family | Use |
|---|---|
| framing | Opening hook, attention box, topic setup |
| focus | Speaker emotion, authority, final takeaway |
| tape/retro | Recalled child interaction, example, quoted situation |
| glitch | Hidden problem, social harm, disruption of naive assumption |
| spotlight | Instruction, key sentence, practical step |
| urgency/sprint | Action step, "do this now" energy |

## Subtitle Rules

Use continuous orange sentence-level subtitles. Segment by complete thought. Do not use word-level captions unless a separate platform style requests it.

## Pacing Rules

Target density for a 2.5-3 minute talking-head video:

| Category | Target |
|---|---|
| visual effects | about 10-14 |
| audio effects | about 13-17 |
| overlays | about 4-6 |
| retained speech segments | about 8-12, after pause/repetition cleanup |

The cue spacing should feel active but not chaotic. If two cues stack on the same moment, one should carry emotion and the other should carry visual focus.
