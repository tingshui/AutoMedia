# Style: 剪映导入-3月6日 v3

## Scope

Vertical talking-head parenting/education opinion videos where the speaker moves from child conflict to parent-facing reframe and practical takeaway.

Status: `needs_review`.

Confidence: medium for videos similar to the 3月6日 reference, low for general creator style.

## Overall Feel

This style is口播 plus light-variety emphasis. Speech remains primary. The edit marks semantic beats in a clear sequence: hook, confusion, wrong instinct, better reframe, example mode, hidden seriousness, hurt, practical method, useful idea, completion.

## Editing Rules

| Rule ID | Rule |
|---|---|
| R1 | Keep speech primary. Every cue must cite a content function. |
| R2 | In the opening hook zone, use a visual framing cue plus a short question/confusion cue when the topic contains conflict or uncertainty. |
| R3 | Mark wrong-parent-instinct moments with error/negative audio. |
| R4 | Mark better-reframe moments with correct/success audio. |
| R5 | In example/recalled-interaction zones, use repeated tape/retro or focus effects at semantic sub-beats, about every 8-12 seconds while the example remains active. |
| R6 | When a hidden social/emotional problem becomes explicit, use tension/shock and a disruption effect if the point lasts longer than one sentence. |
| R7 | When the child's hurt is centered, use sad/broken-family cues and avoid comedy nearby. |
| R8 | At the start of practical advice, restart energy with alert, spotlight, urgency, or tape-style emphasis. |
| R9 | During dense explanation, maintain one light emphasis cue roughly every 8-12 seconds, alternating visual and audio families. |
| R10 | Near the final 15% of the video, use idea/success/completion cues in sequence. |
| R11 | Use overlays briefly on hooks, shock/realization moments, instructions, and final flourish. Overlay duration should usually be 1-4 seconds. |
| R12 | Keep subtitles continuous and sentence-level throughout the speaking section. |
| R13 | Candidate timing must be derived from A semantic zones. Do not reuse exact non-zero reference-answer timestamps; if a time happens to align with a reference boundary, shift to the nearest A-supported semantic boundary and record the reason. |

## Effect Vocabulary

| Family | Use |
|---|---|
| framing | Opening hook, attention box, topic setup |
| focus | Speaker emotion, authority, final takeaway |
| tape/retro | Recalled child interaction, example, quoted situation |
| glitch | Hidden problem, social harm, disruption of naive assumption |
| spotlight | Instruction, key sentence, practical step |
| urgency/sprint | Action step, "do this now" energy |

## Audio Vocabulary

| Family | Use |
|---|---|
| question | confusion, rhetorical question, opening uncertainty |
| error_negative | wrong assumption, bad first instinct, "oh no" moment |
| correct_success | better reframe, answer, action lands |
| shock | realization, seriousness, social boundary violation |
| tension | hidden conflict, escalating emotional stakes |
| sad_broken | child's hurt, emotional consequence |
| variety_beat | explanatory punch, keeps dense口播 lively |
| idea_completion | useful idea, ending resolution, task completion |

## Subtitle Rules

Use continuous orange sentence-level subtitles. Segment by complete thought. Do not use word-level captions unless a separate platform style requests it.

## Pacing Rules

Target density for a 2.5-3 minute talking-head video:

| Category | Target |
|---|---|
| visual effects | about 10-14 |
| audio effects | about 13-17 |
| overlays | about 4-6 |
| retained speech segments | about 8-12, after pause/repetition cleanup |

The cue spacing should feel active but not chaotic. If two cues stack on the same semantic moment, one should carry emotion and the other should carry visual focus.
